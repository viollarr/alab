This component allows you to create new Groups and/or new Access Levels in Joomla! 1.0.15 Stable. 

IMPORTANT - This component includes some hacks for Joomla! 1.0.15 Stable core files.
IMPORTANT - This component does not work with any other versions of Joomla! except Joomla! 1.0.15 Stable. 
	You have to install this component on Joomla! 1.0.15 Stable ONLY.
IMPORTANT - This component does not work with any other components, modules, mambots or hacks as the effects of using them with this component is unknown. 
	It is recommended that you install this component on clean/fresh installation of Joomla! 1.0.15 Stable. 
	For components, modules, mambots or hacks that work with this component, please go to http://www.byostech.com/jaclplus to check.

For questions and support please contact the author at jaclplus@byostech.com or go to http://www.byostech.com/forum.

With this component, you can control your contents to be viewed by groups that created by you. 
To do this, you can create a new group and assign multiple access levels to that group. 
This will be more flexible, and you can decide whether to allow registered users to gain access or not.

NEW INSTALLATION
1) If intall on existing Joomla! 1.0.15 Stable, please BACK UP your files and data FIRST!!!
2) Install com_jaclplus_1.0.15a.zip as a component.
3) If you fail to install this component, please manually fix the permission errors as informed by the installer.

AUTOMATICALLY UPGRADING THRU PATCH COMPONENT (for JACLPlus v1.0.12 or v1.0.13 or v1.0.15 only)
1) Please BACK UP your files and data FIRST!!!
2) Install com_patch_joomla!_n_jaclplus_to_1.0.15a.zip as a component.
3) Uninstall this patch component after complete.
4) If you fail to install this patch component, please manually fix the problems (permission errors) as informed by the installer.

MANUALLY UPGRADING THRU FTP  (for JACLPlus v1.0.12 or v1.0.13 or v1.0.15 only)
1) Please BACK UP your files and data FIRST!!!
Note: This package contain new JACLPlus Configuration Setting (jaclplus.config.php) file. Please backup your jaclplus.config.php file first if you need to refer back.
2) Upload all files in Joomla_1.0.12_to_1.0.15-Stable-Patch_Package_JACLPlus1.0.15a.zip to their respective directories to replace old files. 
3) Change JACLPlus Configuration Setting to suit your need or as previous settings.
